# DevConnect 

O DevConnect é uma plataforma desenvolvida para conectar pessoas iniciando na área de tecnologia a oportunidades de estágio e vagas júnior.

O projeto foi desenvolvido utilizando React no front-end e FastAPI no back-end, permitindo visualizar, cadastrar, editar e remover vagas de forma simples e intuitiva.

Esse projeto foi desenvolvido como parte do processo seletivo da FESF-SUS.

## Funcionalidades

- Visualização de vagas
- Cadastro de novas vagas
- Edição de vagas
- Exclusão de vagas
- Busca por vagas
- Filtro por tecnologias
- Visualização de detalhes da vaga
- Integração entre front-end e API

## Tecnologias utilizadas

Front-end:

- React
- Vite
- JavaScript
- CSS

Back-end:

- Python
- FastAPI
- Pydantic
- Uvicorn
